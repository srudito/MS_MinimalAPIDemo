# MS Minimal API Demo

## 📌 Description
This repository contains a simple ASP.NET Core Minimal API project used for learning and demonstration purposes.  
It showcases how to build lightweight RESTful APIs using the Minimal API approach in .NET.

---

└── projects.sln

---

## 🚀 Getting Started

### ✅ Prerequisites
- .NET SDK (6, 7, or 8)
- Visual Studio or Visual Studio Code

Check installed version:
dotnet --version

---

### ▶️ Run the Application

dotnet restore  
dotnet run  

The API will run on:
https://localhost:<port>  
http://localhost:<port>

---

## 🔍 Testing API

You can test the API using the following options:

### ✅ Requests.http (VS Code)
Open `Requests.http` and click **Send Request**

### ✅ Swagger (if enabled)
https://localhost:<port>/swagger  

### ✅ Postman
Import or manually call the endpoints
